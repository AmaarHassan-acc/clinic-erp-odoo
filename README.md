# Clinic ERP Odoo 18 Community Modules

This repository contains five installable Odoo 18 Community modules for a clinic ERP workflow.

## Modules

Install in this order:

1. `clinic_erp_base`
2. `clinic_erp_schedule`
3. `clinic_erp_clinical`
4. `clinic_erp_billing`
5. `clinic_erp_workspace`

## Features

- Patient records
- Medical staff, rooms, and services
- Appointment scheduling
- Clinical visits and prescriptions
- Billing and invoice workflow
- Reception and doctor workspace menus

## Installation

1. Copy this repository into your Odoo custom addons path.
2. Restart Odoo.
3. Enable Developer Mode.
4. Go to Apps.
5. Click Update Apps List.
6. Search for `Clinic ERP`.
7. Install the modules in the order listed above.
